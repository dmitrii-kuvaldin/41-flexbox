Flexbox ⚡️

Гибкая технология позиционирования элементов в CSS

  • Чтобы "включить" flex отображение в контейнере нужно задать свойство display:flex; для родителя
  • Justify-content задает правила на главной оси
  • Align-items на поперечной оси
  • Justify-content отвечает за положение элементов по оси заданной свойством flex-direction; а algn-items перпендикулярен ему
  • Align-self вы можете применить к элементу ребенку, чтобы изменить его положение по поперечной оси
